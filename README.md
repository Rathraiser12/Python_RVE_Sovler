A custom-built, Python-based 3D Finite Element solver designed specifically for the micromechanical homogenization of Representative Volume Elements (RVEs).

This project was developed from scratch to simulate the non-linear behavior of hyperelastic composite materials (like matrix-fiber structures) and generate rigorous macroscopic stress-strain data.

This project was more for learning how to build a sovler rather than createa perfect and robut rve sovler.
